# Часть 3. Настройка Active Directory

## Задание 1. Развёртывание Active Directory

**Что было сделано:**
- На контроллере домена (Windows Server) установлены роли **Active Directory Domain Services** и **DNS Server**.
- Сервер повышен до контроллера домена с доменным именем `practicumsuperstore.local`.
- В оснастке **Active Directory Users and Computers** создана организационная единица (OU) с именем `IT`.
- Внутри OU `IT` создан пользователь `student` (с указанием ФИО).
- В той же OU создана учётная запись компьютера `PCI-1`.
- В **Group Policy Management** создана групповая политика `IT-policy`, привязанная к OU `IT`.
- В политике настроены параметры интерактивного входа:
  - **Заголовок сообщения:** "Поздравляем"
  - **Текст сообщения:** "Политика успешно применена!"
- Клиентский компьютер `PCI-1` (Windows) введён в домен `practicumsuperstore.local`.
- После входа в систему под пользователем `student` отобразилось приветственное сообщение — политика успешно применилась.
