# Часть 2. Настройка базовых и прикладных сервисов

В этой части выполнены настройки на трёх виртуальных машинах Linux (Хост 1, Хост 2, Хост 3) согласно ТЗ. Все проверки подтверждены скриншотами.

## Задание 1. Настройка служб NTP и DNS

**Что было сделано:**
- На **Хосте 1** (DNS/NTP-сервер) установлены и настроены `chrony` и `dnsmasq`.
- В `chrony.conf` добавлены:
  - `local stratum 10` — сервер выступает как источник времени для локальной сети.
  - `allow 10.10.0.0/16`, `allow 10.11.0.0/16`, `allow 10.12.0.0/16` — разрешены клиенты из всех подсетей проекта.
  - Временная зона установлена на Москву (UTC+3).
- В `dnsmasq.conf` настроен домен `practicumsuperstore.ru` (в скриншоте есть опечатка, но записи созданы для правильного домена):
  - A-записи для `ntp`, `dns`, `mail` (на Хосте 1), `ansible`, `web`, `fs` (на Хосте 2), `mailclient` (на Хосте 3).
  - PTR-записи для обратного разрешения.
  - MX-запись для почтового сервера.
- На **Хосте 2** (Ansible-сервер) настроен клиент NTP и DNS (указан DNS-сервер 10.10.1.86 в `/etc/resolv.conf`).
- Служба `systemd-resolved` отключена (чтобы не конфликтовать с `dnsmasq`).
